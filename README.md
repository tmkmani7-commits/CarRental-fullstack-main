# 🚗 Car Rental & Vehicle Booking Platform

A full-stack **Car Rental and Vehicle Booking Platform** developed as part of the **Skill Wallet Internship Program**.

This application connects **vehicle owners** with **customers** who want to rent cars. Vehicle owners can list their cars on the platform, manage availability, and earn income from rentals, while customers can browse available vehicles, view details, and make bookings through a simple and user-friendly interface.

The project is built using the **MERN Stack (MongoDB, Express.js, React.js, and Node.js)** and demonstrates practical full-stack development skills, including authentication, booking management, database integration, and responsive web design.

### Key Features

#### For Customers

* User Registration and Login
* Browse Available Cars
* Search Cars by Location and Date
* View Vehicle Details
* Book Cars Online
* Manage Personal Bookings

#### For Vehicle Owners

* Add and Manage Cars
* Upload Vehicle Images
* Manage Availability
* Receive Rental Bookings
* Earn Income by Renting Vehicles

#### For Administrators

* Manage Users
* Manage Vehicle Listings
* Monitor Bookings
* Maintain Platform Operations

### Technologies Used

* React.js
* Node.js
* Express.js
* MongoDB Atlas
* JWT Authentication
* ImageKit
* Vite
